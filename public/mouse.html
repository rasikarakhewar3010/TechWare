Mouse

A computer mouse is an input device that allows a user to interact with a computer by controlling the on-screen cursor. It has various forms and functions, with mechanical, optical, and wireless versions available.

History of the Mouse

The computer mouse was invented in the 1960s by Douglas Engelbart. Initially, it was a wooden device with two metal wheels. Since then, it has evolved to include features such as optical sensors, wireless connectivity, and multiple buttons to enhance user interaction.

Types of Mice

1. Mechanical Mouse

Uses a ball to detect motion, common in early computer designs but now largely replaced by optical mice.

2. Optical Mouse

Uses an LED sensor to detect movement on a surface, providing higher precision and durability.

3. Laser Mouse

Similar to the optical mouse but uses a laser for more accurate tracking, especially on smooth surfaces.

4. Wireless Mouse

Connects to a computer wirelessly via Bluetooth or a USB receiver, offering more mobility.

Mouse Components

  • Sensor: Detects movement and translates it to on-screen cursor movement.
  • Buttons: Typically includes left, right, and additional programmable buttons for enhanced control.
  • Scroll Wheel: Allows for vertical scrolling through content.
  • Wireless Transmitter: For wireless mice, connects to a computer via Bluetooth or USB receiver.
